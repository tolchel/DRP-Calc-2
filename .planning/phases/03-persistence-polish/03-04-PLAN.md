  <behavior>
    mergeKDEs(kyberA, competitorA, kyberB, competitorB):
    - Returns array of objects with keys: x, kyberA, competitorA, kyberB, competitorB
    - x values are the union of all x values from all 4 input arrays, sorted ascending
    - For each x, density values are linearly interpolated from the nearest points of each curve
    - Points outside a curve's x range return density 0 for that curve
    - Empty curve array returns 0 for all x values from that curve
  </behavior>
  <action>
Create src/components/ComparisonScreen.tsx. Export mergeKDEs as a named export so ComparisonScreen.test.ts can import it:

```typescript
export function mergeKDEs(
  kyberA: KDEPoint[],
  competitorA: KDEPoint[],
  kyberB: KDEPoint[],
  competitorB: KDEPoint[],
): Array<{ x: number; kyberA: number; competitorA: number; kyberB: number; competitorB: number }>
```

Implement mergeKDEs using the pattern from RESEARCH.md: collect all unique x values (Set + Array.from), sort ascending, then for each x interpolate from each curve using a helper `interpolate(points, x)` that:
- Returns 0 if points is empty
- Returns 0 if x < points[0].x or x > points[points.length-1].x
- Finds the first index i where points[i].x >= x, then linear interpolates between points[i-1] and points[i]
