Run the following sequence in the project root (`/Users/bigpapa/Documents/Vibe_Code/DRP-Calc-2`):

```bash
npm create vite@latest . -- --template react-ts --force
npm install
npm install vite-plugin-singlefile
npm install -D vitest @testing-library/react @testing-library/jest-dom jsdom
```

After scaffolding, update `vite.config.ts` to the exact following content — do NOT use the default scaffold config:

```typescript
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import { viteSingleFile } from 'vite-plugin-singlefile'

export default defineConfig({
  plugins: [react(), viteSingleFile()],
  build: {
    rollupOptions: {
      output: {
        inlineDynamicImports: true,
      },
    },
    assetsInlineLimit: 100_000_000,
    cssCodeSplit: false,
  },
  worker: {
    format: 'es',
  },
  test: {
    environment: 'node',
    globals: true,
    include: ['src/**/*.test.ts'],
  },
})
```

Update `src/App.tsx` to a minimal placeholder (no imports other than React):

```tsx
export default function App() {
  return <h1>DRP Calculator — Loading</h1>
}
```

Update `src/main.tsx` to use React 19 API (createRoot, not ReactDOM.render):

```tsx
import { StrictMode } from 'react'
import { createRoot } from 'react-dom/client'
import App from './App'

createRoot(document.getElementById('root')!).render(
  <StrictMode>
    <App />
  </StrictMode>,
)
```

Do NOT include `import './index.css'` if vite-plugin-singlefile has trouble inlining the default Vite CSS. Keep it if it works; remove if `dist/` produces a separate .css file after build.

In `tsconfig.app.json`, ensure `"strict": true` is set and `"lib"` includes `["ES2020", "DOM", "DOM.Iterable"]`. Do not add `"WebWorker"` here — that will be handled by a separate tsconfig for the worker file in Plan 05.

Note: `useRef()` in React 19 always requires an initial argument. Use `useRef<T>(null)` not `useRef<T>()`. This matters for Plan 05 — record this in code comments.
