## Shared arithmetic helpers (used by both scenarios):

**Total tape throughput:**
```
totalTapeThruMBs = sum(lib.driveCount * lib.driveThroughputMBs for lib in tapeLibraries)
```

**Total data per asset group:**
```
groupDataMB = assetGroup.count * assetGroup.avgSizeGB * 1024
```

**Per-library data (SIM-06):**
```
perLibraryDataMB = totalDataMB / tapeLibraries.length
```

**Network caps (convert Gbps to MB/s):**
```
fastNetCapMBs = fastNetworkGbps * 1000 / 8      // Гбит/с → МБ/с
lanCapMBs     = lanGbps * 1000 / 8
```

## Кибербакап formula (SIM-03):
Direct tape → objects. 10 streams. Speed = totalTapeThruMBs / 10 streams, capped at fastNetCapMBs.

```
effectiveTapeSpeedMBs = min(totalTapeThruMBs / 10, fastNetCapMBs) * networkFactor * uncertaintyFactor
transferMinutes = groupDataMB / (effectiveTapeSpeedMBs * 60)
totalMinutes = sum(transferMinutes + STARTUP_MINUTES for each group in ASSET_PRIORITY order)
```

Groups with count=0 or avgSizeGB=0 contribute 0 transfer time but still add STARTUP_MINUTES (unless count=0, then skip entirely).

## Конкурент formula (SIM-04):
Two-phase: tape → SAN (phase 1), then SAN → objects (phase 2).

Phase 1 — tape → SAN:
```
phase1SpeedMBs = min(totalTapeThruMBs, fastNetCapMBs) * networkFactor * uncertaintyFactor
phase1Minutes = groupDataMB / (phase1SpeedMBs * 60)
```

Phase 2 — SAN → objects (10 streams, capped at LAN):
```
sanStreamSpeedMBs = min(san.streamSpeedMBs, lanCapMBs / san.streamCount)
phase2SpeedMBs = sanStreamSpeedMBs * san.streamCount * networkFactor
phase2Minutes = groupDataMB / (phase2SpeedMBs * 60)
```

```
totalMinutes = sum(phase1Minutes + phase2Minutes + STARTUP_MINUTES for each group in ASSET_PRIORITY order)
```

## Engineer verification (SIM-07 stochastic component — engineerFactor):
engineerFactor = draw from Uniform(0.5, 1.0) — represents availability reduction.
When verifiersPerShift(engineerCount) === 0 → add 480 minutes (8-hour shift pause) per asset group.

## Uncertainty multiplier (SIM-08):
uncertaintyFactor = 1 + Uniform(0, uncertaintyPct) per wave (one draw per asset group per trial).
Applied to transfer time ONLY — not to STARTUP_MINUTES.

## Network factor (SIM-07):
networkFactor = 1 + Uniform(-0.2, 0.2) — same draw shared between Кибербакап and Конкурент in one trial.
Must stay within [0.8, 1.2] by construction.
