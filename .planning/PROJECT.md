# Калькулятор восстановления

## What This Is

Веб-приложение для продавцов систем резервного копирования (Кибербакап), которое позволяет наглядно демонстрировать заказчику преимущество продукта перед конкурентами через сравнение времени восстановления ИТ-инфраструктуры. Пользователь вводит параметры инфраструктуры заказчика, приложение запускает Monte Carlo симуляцию и выводит графики плотности распределения для двух сценариев: Кибербакап vs Конкурент.

## Core Value

Продавец за 5 минут вводит параметры заказчика и получает визуальное сравнение, которое убедительно показывает преимущество Кибербакап по времени восстановления.

- Условия рабочих станций (загрузочный USB и т.п.) — явно помечены в ТЗ как «временно не учитываются»

## Context

- Инструмент для продавцов, запускается на Windows 11 локально в браузере
- Распространяется как собранная папка dist (React + Vite), Node.js нужен только для сборки
- UI полностью на русском языке
- Два сценария сравнения: **Кибербакап** (прямое восстановление с ленты на объекты) vs **Конкурент** (двухфазное: лента → СХД → объекты)
- Восстановление идёт в порядке приоритета: БД → Серверы → Файловые хранилища → Рабочие станции
- Инженеры ИТ работают сменами 8ч, 7 дней в неделю, обеспечивают 24/7 покрытие; влияют на параллелизм фазы запуска/проверки объектов

## Constraints

- **Tech Stack**: React + Vite + recharts + lucide-react — зафиксировано в ТЗ
- **Среда запуска**: Windows 11, браузер, без серверных зависимостей при использовании
- **Симуляция**: min 10 000, max 200 000 триалов Monte Carlo
- **Поправка на неопределённость**: 0–100%, default 15%; применяется к каждой «волне» восстановления
